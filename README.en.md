# Technical Learning Notes Repository

This is a comprehensive technical learning notes repository, consolidating core knowledge and learning insights from popular technology domains such as Java, Python, algorithms, and message queues.

## Main Content

### JavaSE Fundamentals
Covers the core knowledge of Java Standard Edition, including JDK tool usage, environment configuration, common commands, and coding standards—ideal for beginners to systematically learn Java programming.

### LeetCode Algorithm Solutions
Collects detailed problem-solving approaches and code implementations for the top 100 LeetCode problems, covering high-frequency algorithm types such as arrays, linked lists, trees, graphs, and dynamic programming to enhance programming skills and algorithmic thinking.

### Python Programming Guide
Systematically organizes Python learning notes from basic syntax to object-oriented programming, including practical topics such as data types, control structures, functions, classes and objects, and file operations.

### RabbitMQ Message Queue
In-depth exploration of RabbitMQ as a message middleware, covering synchronous/asynchronous invocation, message models, reliability guarantees, delayed messages, and other advanced features—suitable for distributed system development.

### Spring AI Alibaba
Following the AI technology wave, this section focuses on learning the Spring AI Alibaba framework, mastering cutting-edge technologies such as large model application development, conversational interaction, RAG (Retrieval-Augmented Generation), and tool invocation.

## Project Features

- **Systematized Knowledge**: Organized by technology domain to form a complete knowledge network  
- **Illustrated Content**: Includes numerous diagrams and flowcharts to aid understanding  
- **Continuously Updated**: Content is regularly expanded and refined as learning progresses  

## Usage Suggestions

This repository can be browsed and studied online directly. It is recommended to read it in a Markdown editor or any IDE that supports Markdown. Choose relevant sections based on your personal learning needs for in-depth study.
