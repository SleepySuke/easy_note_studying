

# 技术学习笔记仓库

这是一个综合性的技术学习笔记仓库，整理收录了Java、Python、算法、消息队列等热门技术领域的核心知识点和学习心得。

## 主要内容

### JavaSE基础
涵盖Java标准版的核心知识，包括JDK工具使用、环境配置、常用命令、代码编写规范等基础内容，适合初学者系统学习Java编程。

### LeetCode算法题解
收录LeetCode热门100题的详细解题思路和代码实现，涵盖数组、链表、树、图、动态规划等高频算法题型，帮助提升编程能力和算法思维。

### Python编程指南
从基础语法到面向对象编程，系统整理Python学习笔记，包括数据类型、控制结构、函数、类与对象、文件操作等实用内容。

### RabbitMQ消息队列
深入学习RabbitMQ消息中间件，包括同步/异步调用、消息模型、可靠性保障、延迟消息等高级特性，适用于分布式系统开发。

### Spring AI Alibaba
紧跟AI技术浪潮，学习Spring AI Alibaba框架，掌握大模型应用开发、对话交互、RAG检索增强、工具调用等前沿技术。

## 项目特点

- **知识体系化**：按技术领域分类整理，形成完整的知识网络
- **图文并茂**：配有大量示意图和流程图辅助理解
- **持续更新**：随学习进度不断补充和完善内容

## 使用建议

本仓库可直接在线浏览学习，支持在Markdown编辑器或支持Markdown的IDE中阅读。建议根据自身学习需求选择相应章节进行深入学习。

## 技术栈

- 编程语言：Java、Python
- 数据结构与算法
- 消息队列：RabbitMQ
- AI框架：Spring AI Alibaba
